# V15 Single Login + Role Selection

Implemented:
- One login screen only.
- HealthyMe-style centered login card.
- Same login can open Admin or Supervisor access after authentication.
- Role selection appears after login.
- Left Streamlit sidebar is hidden.
- Navigation is inside the page.
- Session-state routing prevents Streamlit rerun/page structure disruption.
- CSS wrappers are used for stable layout on mobile and desktop.
- Supervisor remains ultra-simple with two actions only.
